# RunPod AI Hub Launcher

Open-source AI desktop launcher and infrastructure management tool focused on RunPod workflows, Stable Diffusion environments, serverless AI endpoints, and cloud/local AI management.

## Current Status

Desktop Alpha — Active Development

## Main Features

- RunPod Proxy Awareness
- Dynamic Port Discovery
- SSH Detection
- Health Diagnostics
- Live Download Progress
- HuggingFace Gated Model Detection
- RunPod API Integration
- Serverless Endpoint Support
- Safe Launcher System
- Full English UI
- Desktop EXE Support

## Project Goal

The goal of this project is to simplify AI infrastructure workflows for creators, researchers, and RunPod users by combining cloud AI management, desktop tooling, serverless interaction, and launcher functionality into a single desktop application.
